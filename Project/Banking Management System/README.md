# 💰 Banking Management System (CLI)

A complete **Command Line Interface (CLI)** based Banking Management System built using **JavaScript (Node.js)**.  
This project allows users to manage accounts, perform transactions, and handle loans with EMI tracking.

---

## 🚀 Features

### 👤 User Account Management

- Register new users with validation
- Unique account number generation
- Login using account number & phone number
- Update personal details securely

### 💵 Banking Operations

- Deposit money
- Withdraw money (with balance validation)
- Check account balance

### 🏦 Loan Management System

- Apply for:
  - Home Loan 🏠
  - Business Loan 💼
  - Education Loan 🎓
- Dynamic interest calculation based on duration
- EMI (Equated Monthly Installment) calculation
- Loan status tracking

### 📊 EMI Payment System

- Pay monthly EMIs
- Track remaining EMIs
- Auto-update loan status after completion

### 📋 Account Overview

- View complete user details
- Display active loans
- Show remaining loan duration (months/years)

### 🛠️ Admin Features

- View all registered accounts
